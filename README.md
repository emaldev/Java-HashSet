Java HashSet Exercises

This repository contains my practice sessions focused on understanding and implementing `HashSet` in Java. 

### Contents:
- **NameHashset:** Managing unique collections of names.
- **SpillingWorde:** Filtering duplicate words from a sentence using `split()` and `HashSet`.
- **HashSetchekerz:** Implementing intersection logic between two sets using `retainAll()`.

These exercises are part of my learning journey towards building more complex Java applications,
including game development logic.
